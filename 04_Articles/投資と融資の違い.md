# 投資と融資の違い

**投資**は「仲間になること」、**融資**は「お金を貸すこと」という一言で表現できる。両者の本質的な違いは「返済の義務」と「見返り（リターン）の性質」にある。

## 投資（Investment / Equity）

企業の将来性を見込んで資本金としてお金を出すこと。

- **返済義務**: なし。事業が失敗しても、出したお金を返してもらう権利はない
- **リターン**: 利益が出たときの「配当」や、株価上昇時の「売却益」
- **立場**: 株主（出資者）となり、経営に関与する権利（議決権）を得る
- **特性**: ハイリスク・ハイリターン

## 融資（Loan / Debt / Financing）

銀行などの金融機関から「利息を付けて返す」約束でお金を借りること。

- **返済義務**: あり。事業の成否に関わらず、元本と利息を期限までに返済する
- **リターン**: 提供側（銀行など）にとってはあらかじめ決められた「利息」が利益
- **立場**: 債権者となる。経営に直接口を出す権利は基本的にない
- **特性**: ローリスク・ローリターン（契約に基づいた確実な回収）

## 比較まとめ

| 比較項目 | 投資（Equity） | 融資（Debt） |
| --- | --- | --- |
| **返済の必要** | なし | あり（元本＋利息） |
| **主な提供者** | 個人投資家、VCなど | 銀行、政策金融公庫など |
| **提供側のメリット** | 株価上昇、配当金 | 利息収入 |
| **経営への影響** | 議決権がある場合が多い | 原則としてなし |
| **失敗時の影響** | 投資額がゼロになる可能性 | 借金として残る |

## 資金調達側の視点

- **投資を選ぶ場合**: 返済負担をなくして攻めの経営をしたいとき
- **融資を選ぶ場合**: 経営権を渡さずに一時的な資金を確保したいとき
