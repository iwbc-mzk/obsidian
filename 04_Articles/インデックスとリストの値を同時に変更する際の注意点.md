```Python
A = [1, 2, 3, 4, 0]
s = 0
s, A[s] = A[s], -1
print(s, A)  # -> 1 [1, -1, 3, 4, 0]

A = [1, 2, 3, 4, 0]
s = 0
A[s], s = -1, A[s]
print(s, A)  # -> 1 [-1, 2, 3, 4, 0]
```

インデックスを先に書くと意図した動作にならない。
