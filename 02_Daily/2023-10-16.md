## 2 重配列の 90 度回転を求める

`list(zip(*S[::-1]))` のようにすると 90 度回転した配列が得られる。

```Python
S = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(S)  # [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(list(zip(*S[::-1])))  # [(7, 4, 1), (8, 5, 2), (9, 6, 3)]
print(S[::-1])[[7, 8, 9], [4, 5, 6], [1, 2, 3]]  # [[7, 8, 9], [4, 5, 6], [1, 2, 3]]
```
