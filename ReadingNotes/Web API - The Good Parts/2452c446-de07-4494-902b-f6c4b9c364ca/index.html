    4.6 独自の HTTP ヘッダを定義する¶

    適切なヘッダが存在しないメタデータを送りたい場合は、独自の HTTP ヘッダを定義することができる。新しく定義する場合は、x- という接頭辞を付けて、次にサービスや組織等の名称を付けるのが一般的。
    (例: x-GitHub-Request-Id)
