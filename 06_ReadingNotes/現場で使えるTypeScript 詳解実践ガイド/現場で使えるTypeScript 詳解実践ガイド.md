# なぜTypeScriptなのか？

TypeScript
JavaScriptのスーパーセットとして開発された静的型付けプログラミング言語。
最も重要な追加機能として「型」がある。
ECMAScriptでJavaScriptに実装検討中の機能も先行して実装された。

大規模な開発にはJavaScriptにはいくつかの欠点がある。
- 静的型付けシステムがない
- モジュールシステムがない
  (TypeScript登場当時？現在はモジュールシステムが実装されている)
